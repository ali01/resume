Code

Sphene, NetFPGA Based Internet Router (github, designdoc)

Sphene is a dynamic internet router built on top of the NetFPGA platform. It was built in under ten weeks by Matt Sparks, Sam D'Amico, and myself. Sphene was our submission to a competition run by Nick McKeown's Computer Networking Lab at Stanford.

Our team won first prize for our implementation of OSPF (a dynamic routing protocol) and of GRE tunneling. Sphene's implementation consists of about 14,000 lines of C++ and 3,000 lines of Verilog across 1,300 commits. The competition was judged by Flavio Bonomi, Bob Felderman, Mohan Kalkunte, and Nick McKeown.

I was entirely responsible for Sphene's implementation of OSPF, which is notoriously difficult to get right.

Code sample: ospf_topology.{h, cc}, ospf_router.{h, cc}.

QJAM, Framework for Distributed Machine Learning Algorithms (github, whitepaper)

Many machine learning algorithms are easy to parallelize in theory. QJAM is a Python framework that transparently parallelizes machine learning algorithms that adhere to a MapReduce-like model of computation.

Code sample: R Sparse Autoencoder

Tantalum, UNIX Like Operating System

Tantalum is a UNIX based operating system for the 80x86 architecture. It was built on top of Pintos by Matt Sparks, Juan Batiz-Benet, and myself. The project consisted of ~7,000 lines of C and 650 commits (built in under 10 weeks).

Code sample: git clone git://enrage.me/tantalum.git

Galadriel, Compiler for Java-like Language (github)

Galadriel is a compiler written in C++ for a Java-like language. It makes use of double dispatch via the visitor design pattern to separate the compiler's logic from the syntactic details of each target language (e.g. TAC, MIPS, x86).

Code sample: semantic_analyser.{h, cpp}

Simone, C++ Library inspired by Cheritonian OOP (github)

Simone is a library for C++ that implements a lot of the programming methodology advocated by David Cheriton at Stanford.

Code sample: recursive_mutex.{h}

Python Framework for Locally Weighted Linear Regression (github, whitepaper, slides)
